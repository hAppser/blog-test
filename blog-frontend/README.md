## Frontend Setup

1. **Navigate to the frontend folder**:

   ```bash
    cd blog-frontend
   ```

2. **Install dependencies**:

   ```bash
    npm install
   ```

3. **Run the development server**:

   ```bash
    npm run dev
   ```

4. **Open the application**:
   Visit http://localhost:3000 in your browser to view the application.

   ```bash
    http://localhost:3000
   ```
